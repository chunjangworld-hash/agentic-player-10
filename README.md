## 출품 서비스

### MCP #1 — 효도비서
부모님과 멀리 사는 자녀를 위한 카카오톡 도구.

| Tool | 기능 |
|---|---|
| 안부 한 줄 만들기 | 부모님 컨텍스트 기반 안부 메시지 생성 |
| 의심 메시지 사기 판단 | 보이스피싱·스미싱·사기 패턴 분석 |
| 부모님 프로필 / 일정 | 챙길 일·관심사 관리 |

### MCP #2 — 선물고민러
광고 없는 진짜 칭찬받은 선물을 큐레이션하는 카카오톡 도구.

| Tool | 기능 |
|---|---|
| 큐레이션 | 관계/상황/예산 기반 키워드 + 가격대 추출 |
| 광고 필터 | 협찬·체험단 자동 배제 |
| 카카오 선물하기 연계 | SearchGift로 카탈로그 자동 연결 |
