## 🔍 알아둘 마찰점

1. **Windows + uv 환경**: `python` 명령은 MS Store stub. 실제는 `.venv/Scripts/python.exe` 또는 `C:\Users\11\.local\bin\python3.12.exe`
2. **`.env` 파일은 AnySign4PC에 연결**됨. 더블클릭 X → `notepad .env`로 직접 열기 (메모리에 reference 있음)
3. **PowerShell 새 세션은 PATH 미상속** — `gh`는 `C:\Program Files\GitHub CLI\gh.exe` 절대경로 권장
4. **FastMCP `env_prefix='FASTMCP_'` 환경변수 실제 동작 안 함** — 코드에서 `os.environ.get("FASTMCP_PORT", default)` 명시적 처리 채택
5. **MCP GET 응답 = HTTP 406 + `mcp-session-id` 헤더** — 정상 동작 (Streamable HTTP는 일반 GET 거부)
6. **PR #1 머지 형식 = Squash and merge** — Phase 2.2 PR도 같은 방식 권장
