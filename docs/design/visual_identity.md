## 2. 효도비서 (Hyodo Secretary)

### 마스코트

| 항목 | 내용 |
|---|---|
| **캐릭터** | 도토리를 들고 있는 다람쥐 |
| **상징** | 다람쥐가 도토리를 모으듯, 자녀가 부모님을 챙김 |
| **표정** | 따뜻한 미소 (보살피는 느낌) |
| **자세** | 정중하게 도토리를 내미는 동작 |

### 톤 키워드

- **따뜻함** (warm)
- **귀여움** (cute, but not infantile)
- **신뢰감** (trustworthy — 사기 판단 기능이 있으므로 진중함도 필요)

### 컬러 팔레트 — (가) 따뜻한 자연

| 역할 | 컬러 | HEX (참고) |
|---|---|---|
| 메인 | 베이지 | `#F5E6D3` |
| 서브 | 올리브 | `#9CAF88` |
| 포인트 | 따뜻한 오렌지 | `#E8A87C` |
| 텍스트 | 진한 갈색 | `#4A3728` |

### 카피 톤 예시

```
✅ "오늘 부모님께 안부 한 줄 어떠세요?"
✅ "엄마/아빠가 받으신 카톡, 같이 확인해드릴게요"
✅ "이번 주 챙기시면 좋을 일 모아봤어요"

❌ 너무 격식 차림: "부모님에 대한 효도 행위를 권장드립니다"
❌ 너무 캐주얼: "엄빠한테 톡 ㄱㄱ"
❌ 시혜적: "효도해야 합니다"
```

## 3. 선물고민러 (Gift Curator)

### 마스코트

| 항목 | 내용 |
|---|---|
| **캐릭터** | 선물상자 더미 가운데서 돋보기 든 토끼 |
| **상징** | 토끼 = 영리한 큐레이터, 돋보기 = 광고 X 진짜 찾기 |
| **표정** | 호기심 + 살짝 당황 ("선물 뭐 정하징~ㅠㅠ" 동지 느낌) |
| **자세** | 돋보기로 선물상자들을 살펴보는 동작 |

### 톤 키워드

- **발랄함** (playful)
- **공감** (relatable — "나도 선물 고민해봤어")
- **친구 같음** (friendly — 부담 없음)

### 컬러 팔레트 — (나) 카카오 톤 친화

| 역할 | 컬러 | HEX (참고) |
|---|---|---|
| 메인 | 카카오 옐로우 | `#FEE500` |
| 서브 | 라이트 그레이 | `#E8E8E8` |
| 포인트 | 부드러운 핑크 | `#FFB6C1` |
| 텍스트 | 진한 그레이 | `#3C3C3C` |

### 카피 톤 예시

```
✅ "선물 뭐 줄지 고민이시죠~?"
✅ "광고 없는 진짜 추천만 모았어요!"
✅ "이거 받으면 '센스 있다' 소리 들으실 거예요 ✨"
✅ "마음에 안 드시면 다른 방향 찾아드릴게요!"

❌ 너무 격식: "선물 추천 서비스를 제공합니다"
❌ 너무 진지: "최적의 선물 추천 알고리즘"
❌ 시혜적: "당신을 위한 선택"
```

## 4. 두 MCP 사이의 의도된 대조

| | 효도비서 | 선물고민러 |
|---|---|---|
| **분위기** | 따뜻하고 든든함 | 발랄하고 공감 |
| **사용자 호칭** | "본인" / 격조 있는 부모 호칭 | "님" / 친구 같은 말투 |
| **이모지** | 🌿 🪴 🌰 ☕ | 🎁 ✨ 🔍 🐰 |
| **카피 끝맺음** | "~드릴게요", "~어요" | "~죠~?", "~정하징~ㅠㅠ" |
| **사용자 감정** | 안심·보살핌 | 즐거움·동지애 |

→ 같은 사용자가 둘 다 써도 **다른 기능을 쓰는 느낌** 받음. 카카오 도구함 카드에서 시각적으로 즉시 구분.

## 5. AI 이미지 생성 프롬프트 (Phase 3.1에서 사용)

### 효도비서 마스코트 프롬프트 (Midjourney/DALL-E)

```
Cute friendly squirrel mascot holding a small acorn, warm gentle smile,
soft pastel colors (beige and olive green palette), simple flat design,
minimalist, friendly, transparent background, vector illustration style,
suitable for app icon, korean app branding
```

### 선물고민러 마스코트 프롬프트

```
Cute playful rabbit mascot sitting among colorful gift boxes, holding a
magnifying glass, slightly confused but curious expression (relatable),
kakao yellow and soft pink palette, simple flat design, minimalist,
transparent background, vector illustration style, suitable for app icon,
korean app branding
```

(Phase 3.1에서 실제 생성 후 본인이 선택)

## 6. 적용 위치

| 적용처 | 효도비서 | 선물고민러 |
|---|---|---|
| **PlayMCP 대표 이미지** | 다람쥐 마스코트 (정사각형) | 토끼 마스코트 (정사각형) |
| **카드 UI 배경 컬러** | 베이지 그라데이션 | 노랑 + 핑크 액센트 |
| **카드 헤더 이모지** | 🌿 또는 🌰 | 🎁 또는 🔍 |
| **응답 카피 톤** | 정중·따뜻 (~드릴게요) | 발랄·공감 (~죠~?) |
| **에러 메시지 톤** | 차분한 안내 | 미안한 공감 ("앗 잠시만요~") |
